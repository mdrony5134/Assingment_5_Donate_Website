IMPORTANT FAQ

What is the document object model(DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a tree of objects. It allows developers to interact with and manipulate HTML and XML documents dynamically, modifying elements, styles, and content using JavaScript.

How do you select an element from the DOM?

document.getElementById(id) – Selects an element by its ID.
document.querySelector(selector) – Selects the first element that matches a CSS selector.
document.querySelectorAll(selector) – Selects all elements matching a CSS selector.
document.getElementsByClassName(className) – Selects elements by class name.
document.getElementsByTagName(tagName) – Selects elements by tag name.

What is event delegation in the context of the DOM, why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to manage events triggered by its child elements. Instead of adding separate event listeners to each child, the event bubbles up to the parent, which handles it.
useful:
Improved Performance: Reduces the number of event listeners, especially when handling a large number of child elements.
Dynamic Content: It works for both existing and dynamically added child elements.
Simplified Code: It reduces complexity by consolidating event handling in one place.

How do you manipulate an elements , attributes and styls using the DOM?

element.textContent: Sets or gets the text content of an element.
element.innerHTML: Sets or gets the HTML content of an element.
element.innerText: Sets or gets the text visible in the element.
element.setAttribute(attribute, value): Sets the value of an attribute.
element.getAttribute(attribute): Gets the value of an attribute.
element.removeAttribute(attribute): Removes an attribute from the element
element.classList.add(className), remove, or toggle: Adds, removes, or toggles a CSS class. js
